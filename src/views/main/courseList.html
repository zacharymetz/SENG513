SENG513: Web based applications.
This course is taught by Pavol Federl. Currently at the University of Calgary, there is an academic schedule builder that allows students here to create schedules and generate various combinations of academic schedules with the courses that they are trying to enroll into. However, there does not exist a feature to search for electives that do not require prerequisites specifically. The overall purpose of this application is to provide a student-friendly web platform that allows them specifically access elective courses from the University of Calgary, that do not require any prerequisites. Our goal is to create this web-application to be a useful feature for students. In addition, this platform also aims to provide registrar administrators the option to upload semester course information with ease. This is an area that this application is expected to address. We hope to make the process of selecting quality elective courses simple for students, as well as making the process of uploading new yearly semester elective course information simple for the registrar.
